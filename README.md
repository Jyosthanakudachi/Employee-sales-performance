# Employee Sales Performance Dashboard

## Project Overview

The Employee Sales Performance Dashboard is an interactive Excel-based dashboard designed to analyze and monitor employee sales performance. The dashboard provides insights into total sales, target achievement, regional performance, and individual employee contributions, helping businesses track productivity and make data-driven decisions.

## Features

* Interactive sales performance dashboard
* Employee-wise sales analysis
* Target vs Actual Sales comparison
* Target Achievement (%) tracking
* Away From Target (%) analysis
* Region-wise performance monitoring
* Data visualization using charts and KPIs
* Dynamic reporting using Pivot Tables and Excel functions

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Data Validation
* Conditional Formatting
* Excel Functions (VLOOKUP, IF, SUM, etc.)

## Dataset Information

The dataset contains:

* Employee Code
* Sales Executive Name
* Region
* Daily Sales Records
* Total Sales
* Sales Target
* Target Achievement Percentage
* Away From Target Percentage

## Business Objective

To evaluate employee sales performance, identify top performers, monitor target achievement, and provide actionable insights that support business growth and performance management.

## Key Insights

* Tracked sales performance across multiple regions.
* Identified top-performing and underperforming sales executives.
* Measured target achievement and performance gaps.
* Improved reporting efficiency through interactive dashboard visualizations.
